Hidden Layer 1: Weighted Sum

Ad Spend: $61k

Formula & Calculation

z = x × w + b
Variable definitions: x = input value w = weight b = bias z = weighted sum Neuron 1: z = 0.22 × 1.00 + 0.10 = 0.32 Neuron 2: z = 0.22 × -1.00 + 0.20 = -0.02 Neuron 3: z = 0.22 × 0.80 + (-0.10) = 0.08

Explanation

Each neuron computes a weighted sum: multiply the input by a weight, then add a bias.

What is a weight? A weight controls how much influence the input has on this neuron.

What is a bias? The bias is an offset that lets the neuron fire even when the input is 0.

4 Hidden Layer 1: Weighted Sum

z = x × w + b (weighted sum = input × weight + bias)

Each neuron computes a weighted sum: multiply the input by a weight, then add a bias. Watch how all three neurons in Hidden Layer 1 produce their values from the single input x = 0.22.

Why three? Each neuron learns to detect a different aspect of the input. They start with random weights, then training adjusts them to specialize.

5 Hidden Layer 1: ReLU Activation

a = max(0, z)

ReLU is a yes/no gate. If the weighted sum is positive, it passes through. If negative, it becomes 0 — that neuron is "killed" for this input.

👀 Notice how Neuron 3 in the visualization is shown in red — it was killed by ReLU.

6 Hidden Layer 2: Weighted Sum

z_j = Σ_i (a_i × w_ij) + b_j

Now Layer 2 takes the activations from Layer 1 and combines them with its own weights. Each Layer 2 neuron sees ALL three Layer 1 outputs.

This is where the network starts learning combinations of features rather than just single ones.

Same as before — gate the negative values to zero, let positive ones through. Continue scrolling to see the output...
